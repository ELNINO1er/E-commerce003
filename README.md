# E-commerce003 — L'Or Brun by SCMC

Site e-commerce pour **SCMC** (*Sustainable Commodities Marketing Company*) — marque **L'Or Brun**, cacao ivoirien 100 % naturel (poudre, infusion, masse et beurre de cacao).

## Stack
- Template HTML/CSS/JS (front statique)
- SCSS (`assets/scss`) compilé vers `assets/css`
- Hébergement local : WAMP — `http://localhost/Ecommerce-03/`

## Démarrage
Ouvrir `index.html` via un serveur local (WAMP/Apache) à la racine du dépôt.

## Structure
- `index.html`, `shop-*.html`, `product-detail*.html`, `cart.html`, `checkout.html`, `contact-us.html`, `about-us*.html`, `blog-*.html`…
- `assets/` — `css/`, `scss/`, `js/`, `images/`, `fonts/`, `webfonts/`
